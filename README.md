# xkdc Viewer

This is an infinite scroll comic viewer for xkdc Comics.

- Uses a service worker to work offline
- Installable
- Serverless
- Hosted on a CDN
- Built every 15 min on a CI/CD pipeline
- Built with Next.js, Typescript, React, & TailwindCSS
